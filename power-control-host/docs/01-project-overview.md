# 项目总览

## 项目目的

搭建一套可靠性电源控制上位机，目标包括：

- 集中管理多台 ODP / PSW 电源
- 支持基础监控、上电、下电、循环运行
- 支持后续新增“相对延时”“后上先下”等时序能力
- 支持日志记录和 Excel 导出

## 当前已知情况

- 已有安装包，但没有旧工程源码
- ODP 与 PSW 的编程手册已基本具备
- 现阶段最重要的是先把真实设备联调链路打通

## 当前工程定位

本目录不是最终成品，而是后续开发的起始骨架，重点是：

- 把模块边界先划清
- 把第一阶段准备工作写清
- 让后续代码不至于越写越乱

## 第一阶段需要的最小资源

- 1 台 ODP 电源
- 1 台 PSW 电源
- 1 台 Windows 开发电脑
- 1 个交换机
- 3 根网线
- ODP / PSW 编程手册
- 能长期配合联调的人员

## 第一阶段成功标准

- 网络或接口环境已搭通
- 项目配置可加载
- 能开始对接设备身份识别命令
- 已明确后续通信方式优先级
